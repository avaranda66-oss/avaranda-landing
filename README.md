# A Varanda - Landing Page

Base funcional da landing page de A Varanda, criada para seguir o `DESIGN.md` e os mockups em `mockupexample/`.

## Como rodar

```powershell
npm install
npm run dev
```

Depois abra:

```txt
http://127.0.0.1:3000
```

## Fonte da verdade

- `DESIGN.md`: contrato visual principal.
- `docs/SOURCE_OF_TRUTH.md`: como decidir o que vale quando houver duvida.
- `docs/PROJECT_STATUS.md`: estado atual, feito, pendente e proximos passos.
- `docs/VISUAL_REFERENCES.md`: imagens de referencia do mockup.

## Validacoes

```powershell
npm run design:lint
npm run lint
npm run typecheck
npm test
npm run build
```
